SocialApe is twitter like webapp using react and redux frontend hosted on firebase with Express functions communicating with a firestore database. The webapp also features phone verification via twilio before being able to post a scream! There is also a donation page where payments are accepted via Paypal's API.

SmartBrain

This is a project created using React and Node with a Postgres database deployed on Heroku. Users are able to create account and paste images which is sent to detect if there is a face in the image. If there is a face in the image it will be outlined. Users compete for the higher score by entering the most images!
